## Objective

Membangun sistem Point of Sale (POS) yang:

- Offline First
- Cepat digunakan kasir
- Detail untuk owner
- Mudah dikembangkan
- Menggunakan Supabase + PostgreSQL
- Frontend berupa Progressive Web App (PWA)
